# Contracts

These are the contracts used in examples.
For a more detailed explanation look at `scripts/README.md`.

## Directory structure

```
contracts

├── proofOfReserves: contracts used in the Proof of reserves dApp example

├── PriceAnalytics.sol
├── README.md
└── SimpleFtsoExample.sol
```
